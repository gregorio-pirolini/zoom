# Zoom Effect (Magnifier Lens)

Small front-end demo: hover/touch “magnifier lens” zoom on images.

## Features
- Desktop: hover zoom
- Touch devices: press & hold to activate zoom (no opening new page)
- Uses responsive images (`srcset`) and upgrades to the large original image

## Tech
- HTML + CSS
- jQuery
- Pointer Events (touch)

## Run locally
Just open `index.html` in a browser.
